# Mobile QA

## Viewports

- [ ] Small mobile width checked
- [ ] Typical Android width checked
- [ ] Tablet or wide viewport checked where applicable
- [ ] Landscape behavior checked if supported

## Layout

- [ ] No overlapping text or controls
- [ ] Cards and panels stay within viewport
- [ ] Bottom navigation does not cover critical CTAs
- [ ] Scroll areas work naturally
- [ ] Long labels wrap cleanly

## Product Journeys

- [ ] Onboarding or first-run state works
- [ ] Dashboard loads with synthetic/demo profile
- [ ] Simulator or comparison flows work
- [ ] Calculator flows work
- [ ] Status/checklist flows work where applicable
- [ ] Reset/local storage controls work where applicable

## Device Readiness

- [ ] Icons and splash placeholders render
- [ ] External links open correctly
- [ ] No real sensitive-data collection appears in mobile flows
